COMO USAR

1. Coloque este arquivo index.html na raiz do projeto.
2. Coloque sua pasta assets junto dele, assim:
   /index.html
   /assets/logo-tauros-store.png
   /assets/topo-store-tauros.jpeg
   /assets/qr-pix-unico.jpeg
   etc.

3. Não precisa mudar o nome da pasta assets.
4. O arquivo SQL está separado. Use no Supabase/PostgreSQL para organizar melhor pedidos, usuários, produtos, pagamentos e estatísticas.
